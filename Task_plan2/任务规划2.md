# 学习任务：基于 KUKA 机器人的空间动力学建模与分析

围绕 KUKA iiwa 机器人完成以下任务。

可使用以下任一模型文件作为输入：

* URDF 文件：`Task_plan2/iiwa_description/urdf/iiwa14.urdf`
* XML 文件：`Task_plan2/kuka_iiwa_14/iiwa14.xml`

这两个文件描述的是同一个机器人模型，任选其一即可。

## 任务一：初始构型下的 `Xmat` 与 `Imat` 提取和分析

给定 KUKA 机器人的模型文件，使用 [Pinocchio](https://github.com/stack-of-tasks/pinocchio) 的 API 完成以下内容：

* 提取机器人在**零位构型**下各连杆或关节对应的空间变换矩阵 `Xmat`；
* 提取各刚体对应的空间惯量矩阵 `Imat`；
* 将提取结果与所提供的 MATLAB 文件进行对照，并分析二者在**表达形式**或**数值结果**上存在差异的可能原因。
* 分析 URDF 和 XML 文件中的模型参数是如何映射为 `Imat` 和 `Xmat` 的；

## 任务二：基于初始构型 `Xmat` 的前向运动学分析

已知 KUKA 机器人的各个关节均为旋转副，要求分析以下问题：

* 初始构型下的 `Xmat` 在前向运动学递推过程中扮演什么角色；
* 每个旋转关节的关节坐标变换应如何表示；
* 如何将“初始构型对应的固定变换”与“关节变量引起的运动变换”结合起来，递推得到各关节以及末端执行器的位姿。

## 任务三：RNEA 编码实现

在完成前两项任务并理解相关建模过程的基础上，自主尝试实现 RNEA（Recursive Newton-Euler Algorithm）。

如果能够真正理解前两项任务，那么对 RNEA 的整体思路与计算流程也应能够建立较为完整的认识。
